# docs/

Generated review reports land here.

- `site-review-<date>.html` — output of the `/site-review` skill (five-persona site review: web design, marketing, sales, restaurant owner, accessibility).
- `seo-audit-<date>.html` — output of the `/seo-audit` skill (dedicated SEO audit).

Open the HTML files in a browser; each is fully self-contained. This folder starts as a placeholder so the skills always have somewhere to write on the first run.
